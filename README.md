# Yang-Mills-3D-AI-verification-program
Auditable, MIT-licensed framework for a ledger-gated constructive RG program targeting 3D SU(2) Yang–Mills. A gate DAG, fail-stop scheduler, independently-verified checkers, claim ratchet, and explicit kill/pivot budget keep claims honest.  Not a Clay 4D proof, but an architecture for building a 3D version one gate by gate.
